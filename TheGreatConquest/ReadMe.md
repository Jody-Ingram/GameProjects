# The Great Conquest

**The Great Conquest** is a browser-based 2D platformer and hack-and-slash game inspired by classic Prince of Persia-style adventures.

Fight through an Indian-themed palace, defeat enemy guards, survive traps, defeat the final boss, and claim the treasure.

## Play

Open `the-great-conquest.html` in a modern web browser.

No installation or web server is required.

## Controls

| Action | Control |
|---|---|
| Move Left | `A` or `Left Arrow` |
| Move Right | `D` or `Right Arrow` |
| Jump | `W`, `Space`, or `Up Arrow` |
| Sword Attack | `J`, `X`, `K`, or Left Mouse Click |
| Restart Game | `R` |
| Start Game | `Enter` |

## Objective

Reach the palace vault, defeat the **Iron Vizier**, and collect the treasure.

> **More work is now assigned! Game Over!**
