# TBD — Episode 0.1: The Trial

A self-contained HTML5 canvas game prototype featuring:

- Cinematic opening story crawl
- Side-scrolling 2D platforming
- Mouse-aimed blaster combat
- Double jump with jetpack animation
- Multiple creature enemy types
- Health, particle effects, sound effects, and a final package objective

## Play

Open `index.html` in a modern desktop browser.

## Controls

- **A / D** or **Left / Right Arrow** — Move
- **W / Space / Up Arrow** — Jump; press again in the air to activate the jetpack
- **Mouse** — Aim
- **Left Click** — Fire
- **R** — Restart

No installation or web server is required.
